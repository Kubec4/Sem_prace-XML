Úvod

Semestrální práce z předmětu Značkovací jazyky se zabývá tvorbou XML dokumentu popisujícího soubor do kterého smyšlená aplikace ukládá svá nastavení. Součástí je také tvorba XSD schéma, XSL transformace do JSON a tato dokumentace. Dokument byl tvořen jako konfigurační soubor počítačové hry.

Metodika

Nejprve byl vytvořen hrubý XML dokument se jmény elementů a atributů. Ten byl následně konvertován do XSD pomocí webové aplikace Freeformater v designu Russian Doll (zachovává stejnou strukturu jako zadaný XML dokument). Tento XSD dokument byl pak ručně doplněn o restrikce na datové typy a také byla úpravena celková struktura dokumentu do finální podoby.

Finální XML bylo vygenerováno na základě v předchozím kroku vytvořeného XSD pomocí webové aplikace XSD2XML. Některé hodnoty byly manuálně změněny.

Validita těchto dvou dokumentů byla zkontrolována pomocí webového validátoru . Zároveň bylo zkontrolováno, že XML dokument odpovídá XSD schématu.

Pro XSL transformaci do JSON byl nejprve z XML vygenerován JSON soubor, který byl použit jako základ pro transformaci. Všechny elementy, které obsahovaly pouze jeden atribut a žádný vnořený element, byly přejmenovány podle vzoru element-atribut. Kategorie nastavení vyplívající ze struktury dokumentu byly zabaleny do for-each cyklů, pro zkrácení zápisu adres elementů. Validita JSON souboru vytvořeného transformací byla ověřena na webového validátoru Json Parser Online

Celý projekt byl nahrán na GitHub.

Popis transformace

JSON

Data jsou transformována do JSON dokumentu. Jsou strukturována stejně jako výchozí XML soubor. Elementy s datovým typem string jsou zapsány v úvozovkách. Číselné datové typy a boolean jsou zapsnány bez úvozovek.

Závěr

Jazyk XML je používán pro strukturované ukládání dat nebo při komunikaci mezi programy. Jeho výhodou je dobrá čitelnost a šikoké možnosti filtrování a omezování dat a datovýh typů. Práce s ním je také snadná, protože k jeho editaci stačí pouze textový editor.
