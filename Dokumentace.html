Úvod

Semestrální práce z předmětu Značkovací jazyky se zabývá tvorbou XML dokumentu uchovávájící různá data o uživatelech informačního systemu pro malou obec. Součástí je také tvorba XSD schéma, XSL transformace do HTML a tato dokumentace. Dokument byl tvořen pro informační systém pro malou obec.

Popis bakalářské práce

Cíle práce:

Cílem práce je navrhnout, vytvořit a otestovat informační systém malé obce s méně než 1000 obyvateli ve Středočeském kraji. Systém bude dostupný z běžného programového vybavení na internetu a bude rozlišovat několik úrovní přístupu k informacím: 1) široká veřejnost, 2) pod heslem občan obce, 3) pod heslem člen zastupitelstva obce, 4)moderátor, 5) administrátor. Systém by měl kromě jiného nahradit zákonem danou vyhlášku a veřejnou nástěnku.

Metodika:

Nejprve bude úprovedena analýza stávající infrastruktury v obci a přehled dostupných nástrojů pro vytvoření požadovaného systému. Potom na základě analýzy bude proveden návrh, implemetace a testování.

Popis praktické části

Nejprve se udělá analýza stávající infrastruktury. Vytvoří se první model informačního systému, který se bude dále upravovat pro zisk dalších a lepších verzí informačního systému. Model bude obsahovat informace o občanech, který si v informačním systému vytvoří účet, pro obyvatele dané obce bude účet automaticky předem vytvořen. Pro testování modelu informačního systému bude vytvořeno několik testovacích účtů pro jednotlivé úrovně přístupů, poté budou vytvořeni první pilotní účty fiktivních obyvatel a po dokončení testování se informační systém spustí.

Metodika

Nejprve byl vytvořen hrubý XML dokument se jmény elementů a atributů. Poté byl ručně vytvořen XSD dokument s restrikcemi na datové typy a také byla úpravena celková struktura dokumentu do finální podoby.

Validita těchto dvou dokumentů byla kontrolována pomocí webového validátoru, při téměř každé úpravě . Zároveň bylo zkontrolováno, že XML dokument odpovídá XSD schématu a je "Well Formed" a validní.

Pro XSL transformaci do HTML bylo nejprve určeno co se pro HTML dokument vytáhne z XML souboru. Data, která se z XML vytáhnou jsou: Jméno uživatele, zařazení do skupiny, politická příslušnost, telefoní číslo, adresa a rodné číslo. Poté byla transformace ručně napsána a připojena k XML souboru.

Celý projekt byl nahrán na GitHub.

Popis transformace

HTML

Data z XML souboru jsou transformována do HTML dokumentu. Transformace vybere Jméno uživatele, zařazení do skupiny, politickou příslušnost, telefoní číslo, adresu a rodné číslo. To je následně uspořádáno do tabulky, kde sloupce jsou pro data a řádky pro uživatele.

Závěr

Jazyk XML je používán pro strukturované ukládání dat nebo při komunikaci mezi programy. Jeho výhodou je dobrá čitelnost a šikoké možnosti filtrování a omezování dat a datovýh typů. Práce s ním je také snadná, protože k jeho editaci stačí pouze textový editor.
